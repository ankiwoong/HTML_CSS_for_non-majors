* Hyper Text Markup Language
    * 웹 페이지지를 만드는 언어
    * Hyper Text(링크)
    * Markup Language
    * 확장자가 html

* HTML 문법
    * 태그
    ```html
    <h1>Hello, HTML</h1>
    ```
    * 속성
    ```html
    <h1 id="title">Hello, HTML</h1>
    ```
    * 태그의 중첩
    ```html
    <h1>Hello, <i>HTML</i></h1>
    ```
    * 빈 태그
    ```html
    <br>
    <img src="">
    ```
    * 공백
    ```html
    <h1>Hello, HTML</h1>
    <h1>Hello   HTMML</h1>
    <h1>
        Hello,
        HTML
    </h1>
    ```
    * 주석
    ```html
    <!-- 여기에 작성되는 내용들은 모두 주석 처리가 됩니다. -->
    <!-- 주석은 여러 줄로 작성할 수도 있습니다.
        <h1>Hello, HTML</h1>
        위 <h1>태그는 브라우저가 해석하지 않습니다.
    -->
    ```
