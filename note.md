* Hyper Text Markup Language
    * 웹 페이지지를 만드는 언어
    * Hyper Text(링크)
    * Markup Language
    * 확장자가 html

* HTML 문법
    * 태그
    ```html
    <h1>Hello, HTML</h1>
    ```
    * 속성
    ```html
    <h1 id="title">Hello, HTML</h1>
    ```
    * 태그의 중첩
    ```html
    <h1>Hello, <i>HTML</i></h1>
    ```
    * 빈 태그
    ```html
    <br>
    <img src="">
    ```
